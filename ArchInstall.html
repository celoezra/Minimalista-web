Instalando Arch Linux...

Para deixar o teclado em Abnt2:
loadkeys br-abnt2

Para entrar no editor de disco:
cfdisk /dev/sda
Aqui devemos, começar a configuração e particionamento do sistema.
577M para boot. (EFI)
16GB para Swap (SWAP FILE)
Tem que dividir o sistema.
Quanto você decidir que ficar no /
E quanto você decidir que fica no home
Se for montar uma home separada do /, tem que criar mais um file system, você decide o tamanho.
Formatação:
mkfs.fat -F32 /dev/sda1
mkswap /dev/sda2
mkfs.ext4 /dev/sda3  (/ barra)
mkfs.ext4 /dev/sda4 (home)
Montando:
mount /dev/sda3 /mnt (fica para o barra) (30 GB fica legal, mas eu sempre uso 100)
mkdir /mnt/home 
mkdir /mnt/boot/efi
mount /dev/sda1 /mnt/boot/efi
swapon /dev/sda2
Confira se esta tudo certo com o comando lsblk
Pacman –Sy archlinux-keyring
pacstrap /mnt base base-devel linux linux-firmware nano vim dhcpcd
genfstab -U -p /mnt >> /mnt/etc/fstab
Arch-chroot /mnt -> para entrar no sistema.
ln -sf /usr/share/zoneinfo/America/Sao_Paulo/etc/localtime
hwclock --systohc
nano locale-gen
Descomente o pt-br utf-8
Configure o teclado:
/etc/vconsole.conf
KEYMAP=br-abnt2
Defina a senha super-root
Comando: passwd
Insira a senha e repita quando solicitar.
